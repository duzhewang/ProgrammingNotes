Problem: Ads Performance

MySQL:

```
Select ad_id, if(clicks+views=0, 0.00, round(clicks/(clicks+views)*100, 2)) as ctr from
(Select ad_id, 
sum(case
        when action='Viewed' then 1
        else 0
    end) as views, 
sum(case
       when action='Clicked' then 1
       else 0
    end) as clicks
from Ads
group by ad_id
) as T
order by ctr desc;

```
