- Method 1: use index

```python

class Solution:
    def anagramMappings(self, A: List[int], B: List[int]) -> List[int]:
        return [B.index(num) for num in A]

```

```python

class Solution:
    def anagramMappings(self, A: List[int], B: List[int]) -> List[int]:
        D={char:index for index, char in enumerate(B)}
        return [D[char] for char in A]
```
